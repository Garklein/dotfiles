* Arch Installation Instructions

** Installing Arch

Follow the instructions at https://wiki.archlinux.org/title/Installation_guide.

Make sure you install `networkmanager` when you're chrooted in!

Reboot. Congratulations, you're in Arch.

** Post-boot setup

Create a user:

#+begin_src bash
useradd -m newuser
passwd newuser
#+end_src

If you want to give them sudo, edit `/etc/sudoers`.

Login as the user. Use `su` to install `sudo`, `vim`, `git`, `nix`.

** home-manager (for EXWM etc)

#+begin_src bash
# nix setup
systemctl enable nix-daemon.service
systemctl start nix-daemon.service
nix-channel --add https://nixos.org/channels/nixpkgs-unstable
nix-channel --add https://github.com/nix-community/home-manager/archive/master.tar.gz home-manager # channel for home-manager
nix-channel --update
nix-shell '<home-manager>' -A install
git clone https://github.com/garklein/dotfiles ~/dotfiles
home-manager switch --flake ~/dotfiles#home --extra-experimental-features nix-command --extra-experimental-features flakes
#+end_src
